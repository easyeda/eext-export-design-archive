# 設計アーカイブファイルのエクスポート

## プラグイン概要

これは嘉立創EDA専用の一括エクスポートプラグインであり、プロジェクトファイル、部品ライブラリ、パッケージライブラリなどの設計リソースの一括エクスポートをサポートしています。この拡張機能は特にオフラインクライアントユーザーに適しており、設計リソースをローカルファイルとして簡単に出力し、バックアップや移行を行うことができます。

## 主な機能

このプラグインは以下のエクスポート機能を提供しています：

### 🔧 プロジェクトアーカイブの圧縮ファイルエクスポート

-   指定オーナー下のすべてのプロジェクトを一括エクスポート
-   バージョンアダプティブフォーマット対応
    プロジェクトを開くことなくエクスポート可能
    ![img.png](images/img.png)

`### 📦 デバイスアーカイブのエクスポート`

-   器件ライブラリから一括でデバイスをエクスポート
-   バージョン対応フォーマットサポート
-   ページングエクスポートをサポートし、エクスポートごとの件数を設定可能
    ![img_1.png](images/img_1.png)

### 📋 装置リストのエクスポート (Export Device List)

-   器件ライブラリの詳細リストをエクスポート
-   Excel形式のエクスポートをサポート
    編集リスト後、エディタに内蔵された一括部品作成機能を使用して部品パラメータを一括更新できます
    ![img_2.png](images/img_2.png)

### 🔌 封面圧縮アーカイブのエクスポート

-   封装ライブラリリソースの一括エクスポート
-   データベース分類に基づくエクスポートをサポート
-   一度にエクスポートする数量を設定可能（最大2000件）
-   ページングエクスポートと全量エクスポートの2つのモードをサポート
-   封装ライブラリのバックアップと共有に便利
    ![img_3.png](images/img_3.png)

### 🏷️ 製品出力アーカイブ（立创部品番号に基づく）

-   立创商城番号に基づく部品のエクスポート
-   バージョン対応フォーマットをサポート
-   立創商城の部品ライブラリとの対応が容易
    ![img_4.png](images/img_4.png)
