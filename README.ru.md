# Экспорт файлов дизайна

## Модуль Профиль

Это модуль массового экспорта, разработанный специально для Jiali Creative EDA, который поддерживает массовый экспорт инженерных документов, библиотек устройств, инкапсуляторов и других проектных ресурсов. Это расширение особенно подходит для пользователей оффлайн - клиентов, которые могут легко экспортировать проектные ресурсы для резервного копирования или переноса локальных файлов.

# Главная функция

Модуль предоставляет следующие функции экспорта:

### 🔧  Экспорт пакетов сжатия (Export Project Archive)

- Массовый экспорт всех работ под указанным владельцем
- Поддержка адаптивного формата версий
- Экспорт без открытия проекта.  
  ![img.png](images/img.png)

### 📦  Экспорт пакетов сжатия устройств (Export Device Archive)

- Массовый экспорт устройств из библиотеки
- Поддержка адаптивного формата версий
- Поддержка экспорта страниц, можно установить количество каждого экспорта  
  ![img1.png](images/img_1.png)

### 📋  Список экспортируемых устройств (Export Device List)

- Экспорт подробного списка библиотеки устройств
  Поддержка экспорта в формате Excel
- После редактирования списка параметры устройства могут быть обновлены в пакете с помощью функции создания компонентов с помощью редактора.  
  ! [img2.png](images/img_2.png)

### 🔌  Экспорт пакетов сжатия (Export Footprint Archive)

- Экспорт пакетных ресурсов библиотеки
- Поддержка экспорта по категориям, к которым принадлежит библиотека
- Можно установить количество экспорта за раз (до 2000)
- Поддержка двух режимов экспорта страниц и полного экспорта
- Облегчение резервного копирования и совместного использования хранилищ  
  ![img3.png](images/img_3.png)

### 🏷️  Экспортировать пакет сжатия устройства (согласно установленному порядку) (Export Device Archive Based on LCSC Part No.)

- Экспорт устройств на основе номера торгового центра Личхон.
- Поддержка адаптивного формата версий
- Удобно соотносится с созданием склада оборудования торгового центра.  
  ! [img4.png](images/img_4.png)
