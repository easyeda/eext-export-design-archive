# 匯出設計歸檔檔案

## 挿件簡介

這是一個專為嘉立創EDA設計的批量匯出挿件，支持批量匯出工程檔案、器件庫、封裝庫等設計資源。 該擴展特別適用於離線用戶端用戶，可以方便地將設計資源匯出為本地檔案進行備份或遷移。

##主要功能

該挿件提供以下匯出功能：

### 🔧  匯出工程壓縮包(Export Project Archive)

- 批量匯出指定所有者下的所有工程
- 支持版本自我調整格式
- 無需打開工程即可匯出  
  ![img.png](images/img.png)

### 📦  匯出器件壓縮包(Export Device Archive)

- 批量匯出器件庫中的器件
- 支持版本自我調整格式
- 支持分頁匯出，可設定每次匯出數量  
  ![img_1.png](images/img_1.png)

### 📋  匯出器件清單(Export Device List)

- 匯出器件庫的詳細清單
- 支持Excel格式匯出
- 編輯清單後可通過編輯器自帶的批量創建元件功能進行批量重繪器件參數  
  ![img_2.png](images/img_2.png)

### 🔌  匯出封裝壓縮包(Export Footprint Archive)

- 批量匯出封裝庫資源
- 支持按庫所屬分類匯出
- 可設定每次匯出數量(最多2000個)
- 支持分頁匯出和全量匯出兩種模式
- 便於封裝庫的備份和共亯  
  ![img_3.png](images/img_3.png)

### 🏷️  匯出器件壓縮包(根據立創編號)(Export Device Archive Based on LCSC Part No.)

- 基於立創商城編號匯出器件
- 支持版本自我調整格式
- 便於與立創商城器件庫對應  
  ![img_4.png](images/img_4.png)
